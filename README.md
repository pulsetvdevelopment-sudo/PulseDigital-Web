##  Features

- **Animated Hero Section** with a live particle canvas, key service stats, and direct contact CTAs
- **Content Categories** showcasing 7 service tiers — National Channels, Entertainment, Cinema, News, International, Premium, and Adult content — each with hover card effects
- **Interactive Movie & Series Catalog** with tabbed navigation and a grid layout using real poster images
- **App Showcase Slider** with auto-play, touch/swipe support, and dot navigation using real app screenshots
- **Dynamic Pricing Section** with a duration switcher (1, 3, 6, and 12 months) that updates all plan prices in real time
- **Floating Contact Buttons** (WhatsApp + Telegram) always visible on screen
- **Scroll Reveal Animations** for smooth section entry transitions
- **Fully Responsive** — optimized for desktop, tablet, and mobile
